Display

The display property specifies the display behavior (the type of rendering box) of an element and is the most important element for layout.

Syntax

display: value;

Description

Value Description Example
Inline Displays an element as an inline element (like <span>). Any height and width properties will have no effect Inline 1 Inline 2
Block Displays an element as a block element (like <p>). It starts on a new line, and takes up the whole width available. Block 1 Block 2
Inline-block Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values Inline-block 1 Inline-block 2
